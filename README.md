# Jayyeong Jeong

Frontend developer focused on React-based web services, media-heavy interfaces, and practical deployment workflows.

I like building interfaces that have to work under real schedules: gathering non-developer requests, turning them into screens and features, and shipping changes through deployment.

## Focus

- Frontend: React, JavaScript, HTML/CSS, React Router, Tailwind CSS
- Web service flow: Spring Boot, JPA, MySQL, REST API integration
- Deployment: AWS S3, CloudFront, Route 53, GitHub Actions
- Computer vision experience: Python, OpenCV, YOLOv8 model training

## Featured Projects

### KUAD 2026 ALTEREGO

Archive and store website for the Konkuk University Apparel Design graduation exhibition.

- Built and operated the 2026 site as a solo developer across frontend, backend, DB, and deployment.
- Served a 3-year exhibition archive with 7 teams, 59 designers, portfolio pages, lookbook, runway, and store screens.
- Managed 815 image assets and 14 video assets with responsive layouts, Swiper carousel, WebP assets, and lazy loading.
- Implemented store operations including order creation, order status updates, admin list, soft delete, and Excel download.
- Deployed with AWS S3, CloudFront, Route 53, and GitHub Actions.

Links: [Service](https://kuadarchive.com/) / [Repository](https://github.com/jayyeong/ALTEREGO_2026_VITE)

### Painting Robot Path Generation with OpenCV/CNN

Capstone project for generating painting robot work paths from RGB camera images.

- Worked on data collection, preprocessing, YOLOv8 model training, data augmentation, and model application.
- Used OpenCV-based calibration, stitching, and line detection as part of the image processing pipeline.
- Connected image processing results to a Unity-Python simulation workflow.

Link: [Repository](https://github.com/pnucse-capstone/capstone-2023-1-25)
